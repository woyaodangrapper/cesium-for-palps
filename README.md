**软件名称：脉冲地理信息示例平台(PalpsGIS)** 是基于 [**Cesium**](https://github.com/CesiumGS/cesium) 分叉设计 。目的在于在大型地理信息三维平台中寻找更好的着色器应用方案。这个项目的目标是作为 **CesiumGS/Cesium** 的学习平台。

# 系统建议

- 系统: **Windows 10 1809更新（内部版本17763）** 或最新 / **Windows 11 (任何生成)**
- 架构: **x64/AMD64**
- 显卡: **NVIDIA GeForce GTX 970 或更高 / AMD Radeon RX 570 或更高**
- 联网: **Yes**

# 此项目中使用的第三方存储库和库

- [**Cesium**](https://github.com/CesiumGS/cesium) by CesiumGS
- [**ECharts**](https://github.com/apache/echarts) by Apache

# 如何支持这个项目？

虽然支持并不是义务，但我们始终感激不尽，并因此受到激励，投入更多时间保持该项目及其他项目的活力。因此，如果你决定支持我们，可以通过以下方式进行 😊

- 提供美工指导：帮助我们提升软件的视觉设计和用户体验。
- 提供更多的下载源资料：为我们的用户提供更多可靠的下载源。
- 提供软件翻译：帮助我们将软件翻译成更多语言，以便更多人使用。
- 贡献代码：如果你是开发者，可以提交代码或改进现有功能。
- 提供反馈：分享你的使用体验和建议，以帮助我们改进软件。
